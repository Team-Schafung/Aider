## Functionalities

| Functionality | Description                                              | Tool-kit   | Impairments | Completion |
|--------|----------------------------------------------------------|--------|---------|-----------|
| `Your 3rd eye`   | Allows users to receive audible descriptions of detected objects through the device's camera. | `ReactDOM, CocoSSD` | `Visual` | Yes        |
| `Find nearest object`   | Provides voice commands to locate nearby objects and navigate the surroundings. | `ReactDOM, CocoSSD` | `Visual` | Yes        |
| `Text Identification`   | Converts text images into spoken words, enabling visually impaired individuals to read. | `Tesseract ocr` | `Visual` | Yes        |
| `Read what one says`   | Transcribes spoken words into written text, assisting those with auditory impairments in communicating with others. | `Speech to text` | `Auditory` | Yes        |
| `Custom vibrations`   | Enables users to personalize vibration alerts to specific events or notifications, such as vibrating the phone when the user's name is called. | `Speech to Vibrations` | `Auditory` | Yes        |
| `Your voicebox`   | Converts typed text into audible speech, providing a means of communication for individuals who are unable to speak. | `Text to speech` | `Vocal` | Yes       |
